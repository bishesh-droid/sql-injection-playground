SQL Injection Playground

WARNING: This application is intentionally vulnerable to SQL Injection for educational purposes. DO NOT use this code in production environments. Only run on isolated, authorized systems.

Vulnerable Features:

  • Login Page

    Vulnerable to Login Bypass. Try `admin'--` as username and any password.

  • Product Search

    Vulnerable to Error-based and UNION-based SQL Injection. Try `%' UNION SELECT NULL,sql,NULL,NULL FROM sqlite_master--` to list tables or `%' UNION SELECT NULL,username,password,NULL FROM users--` to dump user credentials.

  • Product Details (Vulnerable)

    Vulnerable to Boolean-based Blind and Time-based Blind SQL Injection via the `id` parameter. Try `?id=1 AND 1=2` for boolean or `?id=1 AND (SELECT sqlite_sleep(5))` for time-based.

Secure Features:

  • Product Details (Secure)

    This endpoint uses parameterized queries to prevent SQL Injection. It is NOT vulnerable.
